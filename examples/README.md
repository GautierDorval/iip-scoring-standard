# Examples (structure only)

Files in this directory are **non-operational** and exist only to illustrate interface shapes.

- They do **not** publish scoring recipes, weights, calibrated thresholds, or operational heuristics.
- Metric values are illustrative and must not be interpreted as calibrated outputs.
- The examples are intentionally minimal and may omit fields that appear in licensed client delivery.

Included examples:

- `iip-report.example.json`
- `qbank.example.json`
- `corpus-snapshot.manifest.example.json`
