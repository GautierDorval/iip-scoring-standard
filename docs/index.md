# Restricted specification index

This repository provides a **restricted, non-operational** specification surface for **IIP‑Scoring™** (Interpretive Integrity Protocol).

The goal is to make the protocol *citable* and *auditable as a reference*, while keeping implementation, calibration, and operational heuristics private.
